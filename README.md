### **Intents**

Anchored, signed, and lineage-tracked.

- Telemetry

- Signature requirements

Concrete, observable behaviors.  
Each UC:
- Declares an intent  
- Defines actor, preconditions, flow, postconditions  
- Attaches traits  
- Emits audit logs and telemetry  
- Forms the basis for agents, firmware, or services
